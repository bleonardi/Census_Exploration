# Census Exploration: Standalone Spatial Analysis

## Overview
A collection of exploratory spatial analyses and data visualizations focusing on US Census Bureau products. This repository serves as a showcase for core data manipulation and mapping techniques used across various larger research projects.

## Key Skills
*   **Tidycensus / Census API:** Programmatic access to ACS and Decennial data.
*   **Thematic Mapping:** Creating choropleth and interactive maps.
*   **Demographic Profiling:** Analyzing population shifts and economic indicators.

## Data Sources
*   **US Census Bureau:** [Official Data Portal](https://data.census.gov/) and [Developer APIs](https://www.census.gov/data/developers/data-sets.html)

## Tech Stack
*   **R / Quarto:** Primary environment for analysis and presentation.
*   **Leaflet:** Interactive web-based mapping.
